# Fonts

Place `ChicagoFLF.ttf` in this directory for the authentic Mac System 1 font.

- **Chicago FLF** — free version of the classic Mac Chicago typeface
  - Download: https://www.dafont.com/chicago-flf.font
  - License: Freeware

If the file is not present, the app falls back to "Press Start 2P" (Google Fonts)
which is loaded via next/font/google in the root layout.
