# Sound Files

Place the following MP3 files in this directory:

| File           | Description                                         | Source suggestion            |
|----------------|-----------------------------------------------------|------------------------------|
| startup.mp3    | Classic Macintosh 128K startup chime                | freesound.org / archive.org  |
| click.mp3      | Mechanical keyboard / mouse click (short, ~100ms)   | freesound.org                |
| crunch.mp3     | Trash crunch / error sound (~500ms)                 | freesound.org                |
| error.mp3      | System beep / error alert                           | freesound.org                |

Until the files are added, the audio system will gracefully silently fail.
